# roam-css-system
Roam CSS System 2.0

# How does this work
- Create or modify your 'roam/css' page.
- Add a new code block and set the language to 'CSS'.
- Add the following line to it:
```@import url('https://rcvd.github.io/roam-css-system/theme-quattro.css');```
